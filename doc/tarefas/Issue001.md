# Issue001: Listagem de Usuários

## Descrição:

Implementar a funcionalidade de **listagem de usuários**, retornando os dados básicos cadastrados no sistema.

## Funcionalidades requeridas:

* Listar usuários cadastrados
* Retornar os campos: **name, email, age**

## Comportamento esperado:

* Ao acessar a rota/tela de listagem:

  * O sistema retorna a lista de usuários
  * Cada usuário contém: **nome, e-mail e idade**
* Não expor dados sensíveis
* Retorno em formato adequado (ex: JSON ou tabela no front)
